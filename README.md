# docs-alfred

## yaml2md

```shell

go install github.com/xbpk3t/docs-alfred/yaml2md@main

```

## gh-merge

用来在gh-actions中合并拆分后的gh.yml，以方便docs-alfred直接调用

```shell

go install github.com/xbpk3t/docs-alfred/gh-merge@main

```
